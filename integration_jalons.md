# Integration des Jalons de recherche

Tout comme les [inventaires](./integration_inventaires.md), les [jalons de
recherche](https://www.arch.be/index.php?l=fr&m=chercheur&r=jalons-de-recherche)
sont une source très précieuse pour débuter une recherche ou guider un
utilisateur qui serait bloqué devant une question de recherche archivistique.
En effet, ces jalons ont été pensés comme des "guides utilisateurs" pour 
s'approprier une question de recherche en particulier. **Ils offrent un scope
plus large que les inventaires** et pourraient donc être utilisés pour réaliser
le 1er niveau d'une recherche sémantique hiérarchique.

Une autre manière de les utiliser pourrait consister à forcer le bot à consulter
systématiquement les jalons afin de détailler un plan d'action à mener pour
compléter la requete d'un utilisateur.

Malheureusement, les jalons ne sont pas tous disponibles publiquement. Mais il
est par contre possible d'en scraper toute une série depuis le
[webshop](https://webshop.arch.be/index2.php?viewpub=pubzoek&clientid=) des
archives (choisir la catégorie "jalons de recherche").
